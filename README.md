# Product Price Calculator

## Description
This project is a simple Python program that calculates the total cost of a product and applies a discount.  
It is a console-based application created to demonstrate basic programming concepts such as functions, user input, and arithmetic operations.

The program receives information about a product, processes the data, and calculates the final amount the user must pay.

## How it works
1. The program asks the user to enter the product name.
2. It requests the quantity of the product.
3. It asks for the unit price of the product.
4. The function `calculate_total()` multiplies the price by the quantity to calculate the total cost.
5. The function `apply_discount()` calculates the discount and determines the final price.
6. The function `show_result()` prints the product details, total price, discount, and the final amount to pay.
